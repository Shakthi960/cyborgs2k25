STRANGERS BATTLE

EVENT RULES

  1. Event Structure: The event will be conducted in two rounds:
    1. Round 1: Techno Box 1v1 Coding Battle – Participants will randomly pick numbers. Those who receive the same number will compete in a 1v1 coding challenge with identical problems. The first participant to pass all test cases qualifies for Round 2.
    2. Round 2: Optimization Arena – Qualified participants will solve unique but equally difficult problems using Python or Java. Winners will be judged based on accuracy, execution time, and optimization.
  2. Allowed Languages:
    1. Round 1: Python, Java, SQL
    2. Round 2: Python, Java
  3. Evaluation Criteria:
    1. Round 1: Fastest participant to pass all test cases.
    2. Round 2: Accuracy, execution speed, and optimized logic.
  4. General Rules:
    1. Each participant competes individually (no teams).
    2. Internet access is strictly prohibited.
    3. Judges’ decisions will be final and binding.
