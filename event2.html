CODE CARNIVAL

EVENT RULES

    Round 1: Tech Titans (MCQ Round)
    • Participants will answer 20–25 multiple-choice questions.
    • Topics include programming (C, Python, Java basics), DBMS, OS, CN, aptitude, and emerging technologies.
    • Time limit: 20 minutes.
    • No negative marking. Each correct answer carries equal weight.
    • Use of mobile phones or the internet during the round is strictly prohibited.
    Round 2: Cryptic Debug (Cryptography + Debugging)
    • Participants will face 4–5 tasks, which include:
      • Debugging code snippets in C, Python, or Java.
      • Solving basic cryptography problems such as Caesar cipher, Base64 encoding, and hash recognition.
    • Time limit: 30 minutes.
    • Partial credit will be awarded for partially correct solutions.
    • Internet access is not allowed; only the provided material/code may be used.
    • Plagiarism or copying from others will lead to immediate disqualification.
    Round 3: DSA Showdown (Coding Round)
    • Participants will solve 5–6 coding problems ranging from easy to intermediate difficulty.
    • Time limit: 45–60 minutes.
    • Evaluation criteria include:
      • Correctness
      • Time complexity and efficiency
      • Code readability
    • Any form of plagiarism or use of the internet will result in disqualification.
