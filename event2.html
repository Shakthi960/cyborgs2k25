CODE CARNIVAL

EVENT RULES

  1. Round 1: Tech Titans (MCQ Round)
    • Participants will answer 20–25 multiple choice questions.
    • Topics include Programming (C, Python, Java basics), DBMS, OS, CN, Aptitude, and Emerging Tech.
    • Time Limit: 20 minutes.
    • No negative marking; each correct answer carries equal marks.
    • Use of mobile phones or internet during the round is strictly prohibited.
  2. Round 2: Cryptic Debug (Cryptography + Debugging)
    • Participants will face 4–5 tasks, including:
      • Debugging code snippets in C, Python, or Java.
      • Solving basic cryptography problems (Caesar cipher, Base64, hash recognition, etc.).
    • Time Limit: 30 minutes.
    • Partial marks will be awarded for partially corrected solutions.
    • Internet access is not allowed; only given material/code can be used.
    • Plagiarism or copying from others will lead to immediate disqualification.
  3. Round 3: DSA Showdown (Coding Round)
    • Participants will solve 5-6 coding problems of varying difficulty (easy to intermediate).
    • Time Limit: 45–60 minutes.
    • Evaluation will be based on:
      • Correctness
      • Time complexity & efficiency
      • Code readability
    • Any form of plagiarism or internet use will result in disqualification.
