EVENT RULES

  1. Event Structure: The event will be conducted in two rounds:
    1. Round 1: ALGO HUNT – Participants solve logic-based puzzles and match them with the correct algorithms from a given set. Strategic Power Cards such as Hint, Swap, and Double Points may be used. Internet access is not allowed.
    2. Round 2: CODE REWIND – Qualified participants solve problems in Python or Java. They will be given an output and must write code that produces the same result using any logic or approach.
  2. Allowed Languages:
    1. Round 1: Not applicable (logic puzzle round).
    2. Round 2: Python, Java.
  3. Evaluation Criteria:
    1. Round 1: Accuracy in matching puzzles with algorithms, effective use of Power Cards.
    2. Round 2: Correctness of output, code clarity, creativity, and efficiency.
  4. General Rules:
    1. Each participant competes individually (no teams).
    2. Internet access is strictly prohibited.
    3. Plagiarism or pre-written code is not allowed.
    4. Judges’ decisions will be final.
