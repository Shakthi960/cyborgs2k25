WELCOME TO THE DIGITAL REVOLUTION

"CYBORGS 2K25, on October 25-26, 2025, is a national-level technical symposium where innovation meets execution. At SRM Valliammai Engineering College, coders, hackers, and tech visionaries converge to push boundaries in AI, cybersecurity, coding, and emerging tech. Through competitive challenges, collaborative hackathons, and cutting-edge workshops, participants will decode complex problems and engineer futuristic solutions. This is more than an event—it’s a launchpad for the next generation of technologists. Ready to build, break, and redefine the future?"

01EVENTS

PAPER NOVA

• Papers should focus on topics related to computer science.
• Teams must submit soft copies of their papers ahead of the event for review.
• Hard copies should be handed over to the judges prior to the presentation.
• Presentations will be followed by a Q&A session, where judges and the audience can engage with the presenters.

CODE CARNIVAL

• MCQ ROUND: Questions from programming languages, databases, OS & current tech.
• CRYPTOGRAPHY & DSA: Solving encryption problems and coding problems based on DSA.

ARTIFY UI

• Participants have to make a web page’s interface for the given problem statement.
• Particpants can use internet and create interface using both Front end programming or Figma/Wix

RECODEX

• Participants will solve logic based puzzels and match each one with correct algorithm for a given set.
• Participants will be given an output and they must write a code that produces the same result using any logic or approach.

STRANGE BATTLE

• 1 vs 1 coding or SQL battles with random challenges.
• Difficulty increases each round.
• Fastest solver wins the Battle title.

BREAKING TECH

• Round 1 – “Brain Freeze” IceBreaker challenge, True OR False, Picture connection, Riddle (Technical)
• Round 2 – “Byte The Ladder”, technical questions based snake & ladder game. Team which answers moves to next round.
